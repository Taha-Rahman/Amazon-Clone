# Amazon Clone (HTML & CSS)

A simple Amazon homepage clone built using pure HTML and CSS, focusing on layout, styling, and responsive UI design.

 Features

Clean and modern Amazon-like UI

Fully responsive layout

Navigation bar, hero section, and product categories

Pure HTML & CSS (no frameworks, no JavaScript)

Beginner-friendly code structure

 Project Structure
/amazon-clone
   ├── index.html
   ├── style.css
   └── images/

 Purpose

This project was created to practice front-end design skills by replicating the core layout and style of Amazon’s homepage.


 How to Use

Download or clone the repository

Open index.html in your browser

Explore the UI

 Technologies Used

HTML5

CSS3
